Key Notation

In this tool, key notation is an experimental short format that represents the key locations on CharaChorder One.


For example, the 'Left Thumb Middle West' key is noted as 'L1bW', and the 'Right Pinky Down (3D press)' is noted as 'R5D'.


The first and the last letters are the shorthands for the side (L/R) and the direction (N/E/S/W/D), respectively.


The middle part indicates the switch. The number represents the finger (1 to 5: thumb to pinky), using the same way as fingering notation for keyboard instruments in modern scores. The additional 'b' and 'bb', originating from the flat (‚ô≠) and double flat (ùÑ´) symbols in music, are used to indicate the secondary/middle and back switch of that finger.
